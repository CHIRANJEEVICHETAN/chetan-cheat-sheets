Git Commands Cheat Sheet

Function: Initializes a new Git repository.

git init

Function: Clones a repository into a new directory.

git clone https://github.com/user/repo.git

Function: Adds file contents to the index (staging area).

git add filename
git add .

Function: Records changes to the repository.

git commit -m "Commit message"

Function: Displays the state of the working directory and staging area.

git status

Function: Updates remote refs along with associated objects.

git push origin main

Function: Fetches from and integrates with another repository or a local branch.

git pull origin main

Function: Downloads objects and refs from another repository.

git fetch origin

Function: Joins two or more development histories together.

git merge branch-name

Function: Lists, creates, or deletes branches.

git branch
git branch branch-name
git branch -d branch-name

Function: Switches branches or restores working tree files.

git checkout branch-name
git checkout -b new-branch-name

Function: Shows the commit logs.

git log

Function: Shows various types of objects.

git show commit-hash

Function: Shows changes between commits, commit and working tree, etc.

git diff

Function: Resets current HEAD to the specified state.

git reset --hard HEAD
git reset --soft HEAD~1

Function: Removes files from the working tree and from the index.

git rm filename
git rm --cached filename

Function: Stashes the changes in a dirty working directory away.

git stash
git stash pop
git stash apply

Function: Creates, lists, deletes or verifies a tag object signed with GPG.

git tag tag-name
git tag -d tag-name
git tag -a tag-name -m "message"

Function: Manages set of tracked repositories.

git remote -v
git remote add origin https://github.com/user/repo.git
git remote rm origin

Function: Shows the history of all commits.

git reflog

Function: Shows the commit logs in one line each.

git log --oneline

Function: Shows the commit logs with a graph.

git log --graph

Function: Resets current HEAD to the specified state.

git reset HEAD file
git reset --hard HEAD~1

Function: Apply the changes introduced by some existing commits.

git cherry-pick commit-hash

Function: Uses binary search to find the commit that introduced a bug.

git bisect start
git bisect bad
git bisect good commit-hash
git bisect reset

Function: Prints lines matching a pattern.

git grep "pattern"

Function: Creates an archive of files from a named tree.

git archive --format=zip HEAD > archive.zip

Function: Verifies the connectivity and validity of objects in the database.

git fsck

Function: Optimizes the repository by cleaning up unnecessary files and optimizing the local repository.

git gc
git gc --aggressive

Function: Creates a bare repository.

git init --bare

Function: Summarizes git log output in a format suitable for release notes.

git shortlog

Function: Creates a new commit that undoes the changes made by an earlier commit.

git revert commit-hash

Function: Removes untracked files from the working tree.

git clean -f
git clean -fd

Function: Reapply commits on top of another base tip.

git rebase branch
git rebase --onto newbase oldbase
git rebase --abort
git rebase --continue

Function: Shows changes between the index and the last commit.

git diff --cached

Function: Resets HEAD to a specific commit, but leaves the index and working tree as they are.

git reset --soft HEAD~1

Function: Resets HEAD and index to a specific commit, but leaves the working tree as it is.

git reset --mixed HEAD~1

Function: Shows what revision and author last modified each line of a file.

git blame filename

Function: Shows various types of objects.

git show commit-hash

Function: Gives an object name based on an available ref.

git describe

Function: Lists all stashes in the stash list.

git stash list

Function: Shows the changes recorded in the stash as a diff.

git stash show
git stash show -p

Function: Applies a stash to the working directory.

git stash apply
git stash apply stash@{0}

Function: Applies a stash and removes it from the stash list.

git stash pop
git stash pop stash@{0}

Function: Removes a single stash from the stash list.

git stash drop
git stash drop stash@{0}

Function: Removes all stashes from the stash list.

git stash clear

Function: Shows a log of changes to the local repository's refs.

git reflog

Function: Shows information about files in the index and the working directory.

git ls-files
git ls-files -s
git ls-files -u

Function: Reads and modifies symbolic refs.

git symbolic-ref HEAD refs/heads/branch

Function: Provide content or type and size information for repository objects.

git cat-file -p commit-hash
git cat-file -t commit-hash
git cat-file -s commit-hash

Function: Applies a patch to files and/or to the index.

git apply patch-file.patch
git apply --index patch-file.patch

Function: Applies a series of patches from a mailbox.

git am patch.mbox

Function: Prunes all unreachable objects from the object database.

git prune

Function: Integrates Perforce branches with Git.

git p4 clone //depot/path@all

Function: Provides a bidirectional bridge between a Subversion repository and Git.

git svn clone svn-repo-url

Function: Instantly browse your working repository in gitweb.

git instaweb [--httpd=webrick|builtin]

Function: Verifies the gpg signature of tags.

git verify-tag tag-name

Function: Shows references in a remote repository.

git ls-remote
git ls-remote origin

Function: Adds, removes, or reads notes attached to objects.

git notes add -m "Note message" commit-hash

Function: Reuses recorded resolution of conflicted merges.

git rerere
