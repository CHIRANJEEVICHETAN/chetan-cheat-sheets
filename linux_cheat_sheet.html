Linux Commands Cheat Sheet

Function: List directory contents.

ls
ls -l

Function: Change directory.

cd /home
cd ..

Function: Print working directory.

pwd

Function: Make directories.

mkdir myfolder

Function: Remove empty directories.

rmdir myfolder

Function: Remove files or directories.

rm file.txt
rm -r myfolder

Function: Copy files or directories.

cp file1.txt file2.txt

Function: Move or rename files or directories.

mv oldname.txt newname.txt

Function: Create an empty file or update the access time of a file.

touch newfile.txt

Function: Concatenate and display file content.

cat file.txt

Function: View file content one screen at a time.

less file.txt

Function: Output the first part of files.

head -n 10 file.txt

Function: Output the last part of files.

tail -n 10 file.txt

Function: Search text using patterns.

grep 'search_term' file.txt

Function: Search for files in a directory hierarchy.

find /home -name file.txt

Function: Change file mode bits (permissions).

chmod 755 file.txt

Function: Change file owner and group.

chown user:group file.txt

Function: Create hard and symbolic links.

ln -s source_file link_name

Function: Report file system disk space usage.

df -h

Function: Estimate file space usage.

du -sh /home

Function: Report a snapshot of current processes.

ps aux

Function: Display tasks and system resource usage.

top

Function: Terminate processes by PID.

kill 1234

Function: Terminate processes by name.

killall firefox

Function: Resume suspended jobs in the background.

bg %1

Function: Bring a background job to the foreground.

fg %1

Function: List active jobs.

jobs

Function: Send ICMP ECHO_REQUEST to network hosts.

ping google.com

Function: Non-interactive network downloader.

wget http://example.com/file.zip

Function: Transfer data from or to a server.

curl -O http://example.com/file.zip

Function: Secure copy (remote file copy program).

scp file.txt user@remote:/path

Function: OpenSSH SSH client (remote login program).

ssh user@remote

Function: File Transfer Protocol client.

ftp ftp.example.com

Function: Simple text editor in the terminal.

nano file.txt

Function: Advanced text editor.

vim file.txt

Function: An interface to the system reference manuals.

man ls

Function: Create an alias for a command.

alias ll='ls -la'

Function: Display a line of text.

echo "Hello, World!"

Function: Set environment variables.

export PATH=$PATH:/new/path

Function: Display or manipulate the history list.

history

Function: Clear the terminal screen.

clear

Function: Archive files.

tar -czvf archive.tar.gz /path

Function: Compress files.

gzip file.txt

Function: Decompress files.

gunzip file.txt.gz

Function: Package and compress (archive) files.

zip archive.zip file1.txt file2.txt

Function: Extract compressed files from a ZIP archive.

unzip archive.zip

Function: Display the amount of disk space available.

df -h

Function: Mount a filesystem.

mount /dev/sdb1 /mnt

Function: Unmount a filesystem.

umount /mnt

Function: Display free disk space.

df -h

Function: Execute a command as another user, typically as the superuser (root).

sudo apt update

Function: Package handling utility for Debian-based distributions.

sudo apt-get install package_name

Function: Package manager for RPM-based distributions.

sudo yum install package_name

Function: Control the systemd system and service manager.

systemctl status ssh

Function: Run a System V init script.

service ssh start

Function: Schedule periodic jobs.

crontab -e

Function: Create a new user.

sudo useradd username

Function: Change a user's password.

sudo passwd username

Function: Modify a user account.

sudo usermod -aG groupname username

Function: Create a new group.

sudo groupadd groupname
